502

Bad Gateway

Server tidak dapat terhubung dengan API eksternal. Koneksi ke sumber data anime terputus.

Kemungkinan Penyebab:

  • API anime pihak ketiga sedang down
  • Server gateway tidak dapat meneruskan request
  • Timeout saat menghubungi API eksternal
  • Konfigurasi proxy bermasalah
  • API endpoint berubah atau tidak valid
